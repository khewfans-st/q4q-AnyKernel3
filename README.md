## Instructions
1. Replace `zImage`.
1. `zip -r9 UPDATE-AnyKernel3.zip * -x .git README.md *placeholder`
1. Reboot into TWRP recovery mode.
1. Copy `UPDATE-AnyKernel3.zip` into phone and install it.
